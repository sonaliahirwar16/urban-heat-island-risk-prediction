# Urban Heat Island Risk Detection

A machine learning-based system that predicts urban heat risk levels using environmental and atmospheric data.

## Technologies Used
Python, Pandas, NumPy, Scikit-learn, XGBoost, Streamlit

## Features
- Predicts heat risk levels (Low, Medium, High)
- Uses pollution and weather parameters
- Real-time prediction using Streamlit
- Provides sustainability recommendations

## Workflow
- Data preprocessing and feature engineering  
- Model training using multiple algorithms  
- Model evaluation and selection (XGBoost)  
- Deployment using Streamlit  

## How to Run

1. Clone the repository:

git clone https://github.com/sonaliahirwar16/urban-heat-island-risk-detection.git

2.Install dependencies:

pip install -r requirements.txt

3.Run the application:

streamlit run app.py
